# firzandz.github.io

Personal portfolio website for **Firzan — Product Designer** with 8+ years of experience in fintech and banking.

Live: [firzandz.github.io](https://firzandz.github.io)

---

## Structure

```
├── index.html          # Main portfolio page (HTML + CSS in one file)
├── assets/
│   ├── icon-frame.svg        # Process step 1 icon
│   ├── icon-explore.svg      # Process step 2 icon
│   ├── icon-iterate.svg      # Process step 3 icon
│   ├── thumb-ovo-nabung.webp # OVO Nabung project thumbnail
│   └── thumb-bridex.webp     # BRIdex project thumbnail
├── case-study/               # (Coming soon) Individual case study pages
│   ├── ovo-nabung.html
│   └── bridex.html
└── README.md
```

## Sections

- **Hero** — Name, availability status, headline, bio, and photo
- **Marquee** — Scrolling skill tags
- **Belief** — Design manifesto
- **How I Work** — 3-step process cards with icons
- **Selected Projects** — Project cards (OVO Nabung, BRIdex Design System)
- **Contact** — Email, LinkedIn, Blog links
- **Footer**

## Tech

- Pure HTML + CSS, no frameworks
- Fonts: Jacquard 24, DM Sans, DM Mono (Google Fonts)
- Responsive: Desktop → Tablet (768px) → Mobile (480px)
- Hosted on GitHub Pages
